Ini adalah InHouse pipeline untuk belajar

Sebelum Running Pipeline, Anda harus menginstall 

1. conda 
2. bwa
3. samtools
4. fastp

Jika Anda tidak memiliki aplikasi tersebut anda dapat menginstall dengan cara
bash Installation.sh

Anda dapat running pipeline dengan cara

bash workflow.sh config.txt (Anda harus mengubah isi di dalam file config.txt sesuai data Anda)
